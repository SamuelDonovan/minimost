# Security Policy

## Reporting a Vulnerability

This is a personal side project maintained in my spare time, so I can't guarantee
a response time, but I take security issues seriously and will do my best to
address them.

If you discover a security vulnerability, please report it privately rather than
opening a public issue:

- Use GitHub's [private vulnerability reporting](../../security/advisories/new)
  (Security tab → Report a vulnerability)

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested fix, if you have one

## Scope

This project is provided as-is. There is no bug bounty program.

## Supported Versions

Only the latest version on the `main` branch is supported.
