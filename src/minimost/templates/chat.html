Messaging

  • Enter – Send message
  • Shift + Enter – New line
  • Hover a message to reply, edit, delete, or react with emoji

Navigation

  • i – Focus message input
  • o – Start new DM
  • / or f – Search messages
  • j / k – Scroll down / up
  • d / u – Scroll down / up (2×)
  • G / gg – Jump to bottom / top
  • Ctrl J / Ctrl K – Next / previous channel
  • Esc – Close menus / unfocus input
  • ? – Open this help

Visual Mode

  • v – Enter visual mode (highlights most recent message)
  • j / ↓ – Move selection to next message
  • k / ↑ – Move selection to previous message
  • d – Delete highlighted message
  • c – Edit highlighted message
  • o – Reply to highlighted message
  • y – Copy highlighted message text to clipboard
  • e – React to highlighted message with emoji
  • Esc – Exit visual mode

Text Formatting

  • Ctrl B – Bold
  • Ctrl I – Italic
  • Ctrl U – Underline
  • Ctrl S – Strikethrough

Media & Display

  • Paste, drag, or use the paperclip to attach images
  • Images are periodically deleted to save space
  • Pinch or Ctrl + Scroll to adjust font size

Status Indicators

  • Active – Currently using the app
  • Idle – Tab open but no recent activity
  • Away – Tab hidden or in background
  • Offline – Not connected
