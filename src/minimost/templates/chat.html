Messaging

  • Enter – Send message
  • Shift + Enter – New line
  • Hover a message to reply, edit, delete, or react with emoji
  • A ✓ on your own message means it has been read by at least one other person
  • Hover over the check to see who has read your message
  • Messages older than the server's retention period are automatically and permanently removed

Navigation

  • i – Focus message input
  • o – Start new DM
  • / or f – Search messages
  • j / k – Scroll down / up
  • d / u – Scroll down / up (2×)
  • G / gg – Jump to bottom / top
  • Ctrl J / Ctrl K – Next / previous channel
  • Esc – Close menus / unfocus input
  • ? – Open this help

Visual Mode

  • v – Enter visual mode (highlights most recent message)
  • j / ↓ – Move selection to next message
  • k / ↑ – Move selection to previous message
  • d – Delete highlighted message
  • c – Edit highlighted message
  • o – Reply to highlighted message
  • y – Copy highlighted message text to clipboard
  • e – React to highlighted message with emoji
  • Esc – Exit visual mode

Text Formatting

  • Ctrl B – Bold
  • Ctrl I – Italic
  • Ctrl U – Underline
  • Ctrl S – Strikethrough

Calls

  • Click the phone icon in a DM or private channel header to start a call
  • Accept or reject incoming calls from the notification banner
  • During a call: toggle microphone mute, camera on/off, and screen sharing
  • Screen share takes over the main view; camera shrinks to a corner
  • Stopping your browser's screen capture automatically ends the share

Channels & DMs

  • Click + next to Direct Messages to start a DM with one or more people
  • Click × on a DM to hide it from the sidebar — it reappears when a new message arrives
  • Click + next to Channels to create a private invite-only channel
  • Private channel members can add or remove people and rename the channel

Media & Display

  • Paste, drag, or use the paperclip to attach any file type
  • Images display inline; other files appear as a download link
  • Maximum attachment size: {{ max_upload_mb }} MB per file
  • Attachments are periodically deleted to save space; a File deleted label appears in their place
  • Pinch or Ctrl + Scroll to adjust font size

Settings

  • Click your username in the header to open Settings
  • Choose a display name colour from the palette
  • Upload or remove a profile avatar (max {{ max_avatar_mb }} MB)
  • Click the bell icon in the header to mute or unmute notification sounds

Status Indicators

  • Active – Currently using the app
  • Idle – Away or no recent activity
  • Offline – Not connected
